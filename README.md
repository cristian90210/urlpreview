# README #

Get information from a url to use as a preview

### What is this repository for? ###

* Get information from a url to use as a preview
* Version 1.0

### How do I get set up? ###

* Dowload
* Execute npm install
* Execute npm run start
* Listen on port 3000
* Post service on /link-preview
* Use Json params:
{
    "url": "url-get-info"
}

### Who do I talk to? ###

* Repo owner or admin
* Other community or team contact
